Das ist der HCI-Abschluss Root Folder
./game_original			enthählt die Originale kaum angepasste Version des games
./Ideen für Controlle	Ideen es Controllers
./pacman				Finale Game
./pacman/Arduino		Arduino Code des Projektes
./pacman/NodeServer		Auf dem Computer ausgeführter code
./Tagebücher			Die Tagebücher, Das .docx enthählt Videos und sollte mit Word2016 geöffnet werden
./Präsentation.html		Downloadlink der Präsentation, Aufgrund der Größe ausgelagert
./Video.html		 	Downloadlink des Videos, Aufgrund der Größe ausgelagert

Zum starten des Spieles:
Arduino an Pc anschliesen
Arduino Code auf den Arduino laden
Readme in ./pacman/NodeServer für den Start des Servers befolgen
Seite im Browser öffnen: localhost:30001

Mit Schritt-Feature: localhost:30001?steps=<Am tag geleistete Schritte>

Bei Fehlerhaften verhalten hilft mehrmaliges Reseten des Arduinos bzw. neustart des Servers/Neuladen der Webpage
